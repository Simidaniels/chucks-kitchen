# Chucks Kitchen

A Modern restaurant for chucks kitchen, showcasing authentic Nigerian home-style meals with a clean, responsiveness user interface. Built using React + Vite with custom CSS styling and reusable components.


## Tech Stack Used

** React (Frontend Framework)
** Vite (Build Tool)
** CSS (Custom Styling)
** React Icons

## Responsiveness

** Desktop --> split layout
** Tablet --> stacked sections
** Mobile --> vertical layout
